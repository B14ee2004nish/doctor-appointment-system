# Doctor Appointment System

A web-based Doctor Appointment System developed using **Python** and **Flask**. This application allows patients to register, log in, browse available doctors, and book appointments online. It also provides an admin panel to manage doctors and appointments.

## Features

- User Registration
- User Login
- View Doctors
- Doctor Details
- Book Appointments
- Patient Dashboard
- Admin Dashboard
- Appointment Management

## Technologies Used

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- SQLite
- SQLAlchemy

## Installation

1. Clone the repository:

```bash
git clone https://github.com/B14ee2004nish/doctor-appointment-system.git
```

2. Go to the project folder:

```bash
cd doctor-appointment-system
```

3. Install the required packages:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
python app.py
```

5. Open your browser and visit:

```
http://127.0.0.1:5000
```
